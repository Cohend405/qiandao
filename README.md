# qiandao
河南牧业经济学院 今日校园 签到脚本

 1.安装python环境
 
 2.下载config.yml和index.py文件
 
 3.修改config.yml用户配置文件（如果你老师发的普通签到任务，只填写用户名和密码即可；如果任务有签到问题，根据自己手机以前的签到记录自行修改config.yml中的问题和答案）
 
 4.运行index.py签到程序
